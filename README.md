# Method Lab

Corporate website developed for a consulting company focused on business process automation and workflow optimization.

## Live Demo

https://methodlab.mx

## Overview

Method Lab is a corporate website designed to showcase the company's services, methodology, and automation solutions for businesses.

The project combines a dynamic visual experience with a scroll-based navigation system, where sections progressively stack on top of each other to create an engaging and immersive browsing experience.

In addition, it integrates a custom lead capture workflow through a Node.js backend connected to the company's CRM platform.

## Features

- Fully responsive design.
- Immersive single-page experience with stacked scroll transitions.
- Contact form connected to a Node.js backend.
- Monday CRM integration.
- Scroll-triggered animations.
- Interactive content carousels.
- Embedded video content.
- SEO optimization using React Helmet.
- SVG-based icons and graphic assets.
- Modular architecture designed for maintainability and scalability.

## Architecture
